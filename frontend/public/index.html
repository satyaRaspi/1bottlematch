Truflux Bottle Signature Core

3-axis physical signature generation and tolerance-based bottle identification

v1.6.0

Controlled Capture Profile

This version assumes all bottle photos are captured at the same height, same camera distance, same background, same lighting and same 3-axis setup. Geometry remains the primary hard-gate, while ML assists feature extraction through optional ONNX segmentation, color clustering, texture, gradient profile and cross-view consistency.

Core Functions

1. Build Signature

Capture front, side and top photos to generate a physical bottle signature.

2. Register Bottle

Store brand, SKU and the master signature in the signature repository.

3. Identify Bottle

Recreate the 3-axis signature during inspection and match using strict controlled geometry, score, color, category, primary identity and ML-assisted feature gates.
