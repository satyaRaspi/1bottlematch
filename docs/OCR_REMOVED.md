# OCR Removed — v1.6.0

OCR is fully removed from this clean build.

Removed:

- pytesseract dependency
- Tesseract system dependency
- EasyOCR dependency
- OCR registration preview
- OCR text gate
- OCR text comparison
- `/ocr/status`
- `/signature/text-preview`

No OCR or Tesseract installation is required.
