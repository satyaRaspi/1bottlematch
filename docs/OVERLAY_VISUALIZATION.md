# Overlay Visualization — v1.5.7

This version generates visual overlays for each 3-axis bottle image.

What is shown on overlay images:
- Green segmentation mask
- Yellow bottle contour
- Orange bottle bounding box
- Cap region
- Label region
- Body region
- Measurement reference lines

Where shown:
- In the Processing Bottle Match dialog
- In stored Processing Bottle Match records
- On registered bottle detail records through stored image assets
